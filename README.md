A simple weblog website built using Laravel.

## Features

- UserPanel & AdminPanel With Laravel Filament
- Role Management with Laravel Shield
- Site Setting (Admin Panel)
- Articles (UserPanel & AdminPanel)
- About Us
- Contact Us

- **MYSQL** - Database
